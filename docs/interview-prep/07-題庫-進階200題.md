## 脈絡 I — Java 語言與 JVM 深化（201–228）

**簡化講解：** javac 編譯 → bytecode → 類載入（載入→驗證→準備→解析→初始化）→ 解釋/JIT 執行；JIT 探測熱點代碼並編譯機器碼、C1 快速 vs C2 優化；雙親委派防竄改（Bootstrap→Platform→App ClassLoader 逐層問）；物件記憶體佈局：Mark Word(偏向鎖/輕量級鎖標記) + 類別指標 + 實例資料 + 填充；逃逸分析可棧上分配/標量替換/鎖消除；GC 分代：Minor GC（新生代滿）→ Major（老年代滿）→ Full（兜底+清理）；三色標記併發漏標用 SATB/增量更新；G1 Region 化+可預測停頓；ZGC 著色指標+讀屏障+幾乎全併發；引用類：Strong（保活）/ Soft（快取）/ Weak（WeakMap 防洩漏）/ Phantom（追蹤回收）；Full GC 頻繁排查：GC log→jstat→jmap+MAT；OOM 五種（heap/Metaspace/GC overhead/direct memory/request memory）；final 欄位建構完成保證可見性；String.intern() Java 7 後進 heap 自動池；Java 只值傳遞（引用拷貝）；Integer 快取 -128~127、大迴圈自動裝箱浪費；拋例外 fillInStackTrace 成本高；try-with-resources 逆序關閉+suppressed；反射 Method.invoke 慢（跳過 JIT）；註解 SOURCE/CLASS/RUNTIME 保留時機；虛擬執行緒輕量（掛載/卸載 carrier thread）、synchronized 釘住；record/sealed/pattern 搭配做 Kafka 事件不可變。
