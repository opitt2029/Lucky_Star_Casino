# 遊戲規則規範（以前端呈現為準）

> 最後同步：2026-07-20
>
> 本文件整理玩家在前端目前看得到、操作得到的遊戲規則。若前端新增玩法、提示、魚種、側注、障礙效果或賠付說明，先同步本文件；若該規則會影響實際派彩、扣款、RTP 或 Provably Fair 重放，還必須同步後端權威實作、`contracts/*.json`、mock 與測試。

## 同步原則

1. 前端規則卡、魚種資訊卡、下注面板、側注面板與捕魚 Canvas 內提示，是玩家可見規則的來源。
2. 會影響真實結算的數值，必須同步到 `backend/game-service` 與 `contracts/*.json`。
3. 只在前端呈現、尚未進入後端結算的功能，需標記為「展示 / 追蹤規則」，不得寫成會扣款或派彩。
4. 前端 mock 預設會影響玩家本機體驗，仍需跟契約檔或後端邏輯保持一致。

## 老虎機 Slot

### 下注與流程

- 下注選項：`100`、`500`、`1,000`、`MAX`。
- `MAX` 依目前餘額取最高可下注額，但單局上限為 `5,000` 星幣，最低下注為 `100` 星幣。
- 餘額不足時不能轉動。
- 派彩金額為含本金口徑，轉輪動畫結束後同步更新餘額。

### 中獎線與賠付

- 只判定中線。
- 中線三連相同為主要派彩。
- 中線左二格相同且第三格不同時，依符號給小獎。
- 右二同、非中線連線與其他盤面組合不派彩。

| 符號 | 左二同倍率 | 三連倍率 |
|---|---:|---:|
| CHERRY | 1x | 5x |
| LEMON | 1x | 8x |
| BELL | 2x | 18x |
| STAR | 3x | 40x |
| SEVEN | 5x | 70x |

## 百家樂 Baccarat

### 基本下注

- 可下注區域：閒家、莊家、和局。
- 單局總下注範圍以前端為準：`100` 至 `5,000` 星幣。
- 玩家可使用籌碼盤選擇下注額、加倍或清除下注。
- 餘額不足或下注額超出範圍時，不可發牌。

### 點數與補牌

- A = 1 點，2~9 依牌面點數，10 / J / Q / K = 0 點。
- 手牌點數取總和個位數。
- 任一方前兩張為 8 或 9 點時為天牌，雙方不補牌。
- 閒家 0~5 點補牌，6~7 點停牌。
- 莊家依標準 Punto Banco 補牌表補牌。

### 派彩與返水

- 閒家勝：1 : 1。
- 莊家勝：0.95 : 1，含 5% 傭金。
- 和局勝：8 : 1。
- 和局時押莊 / 押閒為 push，退回本金。
- 每局依前端呈現保留 0.5% 返水規則。

### 新增前端規則：側注追蹤

目前前端有側注追蹤面板，但它是「展示 / 追蹤規則」，不進入後端下注 payload，也不影響 wallet 實際扣款或派彩。

| 側注 | 顯示賠率 | 前端判定 |
|---|---:|---|
| 閒對 | 11 : 1 | 閒家前兩張 rank 相同 |
| 莊對 | 11 : 1 | 莊家前兩張 rank 相同 |
| 完美對 | 25 : 1 | 任一方前兩張 rank 與 suit 都相同 |
| 任意對 | 5 : 1 | 任一方前兩張 rank 相同 |
| 大 | 0.54 : 1 | 本局總牌數為 5 或 6 張 |
| 小 | 1.5 : 1 | 本局總牌數為 4 張 |
| Super Six / 超級六 | 12 : 1 | 莊家以 6 點獲勝 |
| Dragon Bonus | 最高 30 : 1 | 非和局且勝負點差至少 4 點 |

### 新增前端規則：咪牌 / 展示流程

- 前端支援咪牌模式與逐張發牌動畫。
- 咪牌與動畫只影響揭牌節奏，不改變後端已決定的牌局、勝負或派彩。

## 捕魚機 Fishing

### 入場、彈藥與結算

- 進入捕魚機頁面時先顯示 buy-in 畫面。
- 入場金額範圍：`100` 至 `1,000,000` 星幣。
- 場中局內餘額不足時可開啟臨時加值，加值畫面置中於遊戲畫面。
- 每次開火會消耗一發彈藥金額。
- 前端彈藥選項：
  - 銅 / 輕型：`10` 星幣 / 發。
  - 銀 / 標準：`50` 星幣 / 發。
  - 金 / 重型：`100` 星幣 / 發。
- 彈藥 / 砲台在 buy-in 前選擇，入場後本局固定。
- 本局盈虧在每次 buy-in 後歸零，結算時顯示淨收入。

### 命中、捕獲與提示

- 魚有 HP，子彈造成傷害；HP 歸零那發會進入捕獲判定。
- 捕獲成功才會取得實際派彩；派彩以後端結果為準。
- 擊倒但未捕獲時，魚會掙脫逃跑，無派彩。
- 遊戲頂部會顯示所有擊倒 / 捕獲 / 障礙魚擊破提示。
- 財神與搖錢樹的頂部提示描述特殊獎勵效果，不顯示固定星幣價值。

### 魚種與倍率呈現

| 分類 | 前端呈現 | 規則 |
|---|---|---|
| 小型魚 | 錦鯉、金魚、燈籠魚 | 低 HP、低倍率、出現頻率高 |
| 中型魚 | 河豚、神仙魚、魔鬼魚 | 中等 HP 與倍率 |
| 高分魚 | 金龍、貔貅 | 高倍率、較稀有、藍色高級特效 |
| 特殊魚 | 財神爺、搖錢樹 | 金色特殊魚特效；財神顯示高倍率獎勵，搖錢樹捕獲時倍率浮動 |
| Boss | 金星魚王、彩金魚王 | 紫色 / 傳奇特效；彩金魚王為前端高價外觀，實際結算仍以後端 DRAGON_KING 結果為準 |

### 搖錢樹規則

- 捕獲後會在遊戲畫面右下角顯示 15 秒效果倒數。
- 搖錢樹是特殊魚。
- 捕獲時倍率在 `10x` 至 `50x` 區間內浮動。
- 前端資訊卡不顯示固定星幣價值，改顯示特殊獎勵。

### 財神規則

- 捕獲後會在遊戲畫面右下角顯示 15 秒效果倒數。
- 財神在前端視為特殊魚。
- 游動時使用金色特殊魚效果。
- 捕獲成功提示為高倍率星幣獎勵，不顯示固定星幣價值。
- 實際派彩仍以後端捕獲結果為準。

### 障礙魚規則

障礙魚射擊會消耗本局彈藥，不派彩，不參與殘血回收。擊破後在遊戲頂部顯示對應效果提示。

| 障礙魚 | 擊破條件 / 呈現 | 效果 |
|---|---|---|
| 障礙章魚 | 大型，約 5 發擊破 | 噴墨遮蔽漁場視野約 2 秒 |
| 障礙海星 | 大型，約 5 發擊破 | 目前魚群加速約 2 秒 |
| 障礙海龜 | 小 / 中 / 大尺寸 | 約 5 / 10 / 17 發擊破，觸發裝甲破裂效果 |

### 前端視覺與操作規則

- 捕魚場景由 PixiJS Canvas 呈現，魚、子彈、光圈、光點、浮字與提示都在 canvas 內繪製。
- 遊戲畫面隱藏瀏覽器游標，只保留遊戲準星。
- 魚離開畫面後才正式移除，普通網頁版與全螢幕版都需一致。
- 高分魚為藍色特效、Boss 為紫色特效、特殊魚為金色特效；光圈在魚背後，不覆蓋魚身。
- buy-in 畫面開始播放海洋風背景音樂，音量由設定控制。
