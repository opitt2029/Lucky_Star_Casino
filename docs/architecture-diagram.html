Lucky Star Casino · 系統架構

幸運星幣城 — 微服務系統架構圖

模擬幣線上娛樂平台。所有外部流量經 API Gateway 統一入口，路由至 7 個業務微服務；服務間以 Kafka 事件解耦、以內部 REST 同步扣款，資料採 PostgreSQL（帳務寫）＋ MySQL（查詢讀）CQRS 分離。
